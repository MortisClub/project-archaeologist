# Как внести вклад

Проект небольшой, так что без лишних формальностей.

## Собрать и проверить

```
cargo build
cargo fmt --check
cargo clippy --all-targets -- -D warnings
cargo test
```

Ровно это же гоняет CI, так что если локально всё зелёное — на GitHub тоже будет.

## Пул-реквесты

Одна ветка — одна тема. В описании коротко: что меняется и зачем. Перед отправкой
прогоните три команды выше.

## Что особенно пригодится

- граф импортов для новых языков (сейчас только JavaScript/TypeScript и Python);
- настоящий treemap в HTML-карте вместо полос.

Нашли баг — заводите issue и приложите, на каком проекте воспроизводится (язык, есть ли git).
